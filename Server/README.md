# SamSWAT's Anime Sight 4.0 Port
This mod allows you to use SamSWAT' Anime Sights on SPT >=4.0.0.

## Installation
1. Download the zip file
2. Extract in game folder
3. Open game
4. (Optional) Be a dirty weeaboo
5. Profit
